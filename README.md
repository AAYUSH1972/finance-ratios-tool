# finance-ratios-tool

A simple tool to calculate key financial ratios for analysis.

## Features
- Calculates Return on Equity (ROE)
- Calculates Current Ratio
- Easy-to-use functions
- Beginner-friendly code

## Example Usage
You can input basic financial data and calculate ratios like:

- ROE = Net Income / Shareholder Equity  
- Current Ratio = Current Assets / Current Liabilities  

## Purpose
This project is built as a learning tool for financial analysis and CFA concepts. It helps understand how basic financial ratios are calculated using simple code.
